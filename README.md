(AI Lab 12 — Flask Introduction)

This project implements the **AI Lab 12** Flask introduction using a **web-based Tic Tac Toe** game.

## What you get
- A Flask app (`app.py`) with a homepage route `/`
- Frontend UI built with:
  - `templates/index.html` (HTML + Jinja2)
  - `static/css/style.css` (glowing aesthetic)
  - `static/js/script.js` (game logic)

## Run locally
```bash
pip install -r requirements.txt
python app.py
```
Then open:
- http://127.0.0.1:5000
